# Godogen — Talent Market Format

This repository is a [Talent Market](https://one-man-company.com) conversion of **Godogen**, an autonomous AI game development pipeline that builds complete Godot 4 projects from natural language descriptions.

## Original Source

> **Repository:** [https://github.com/htdt/godogen](https://github.com/htdt/godogen)
>
> **Author:** Alex Ermolov ([@alex_erm](https://x.com/alex_erm))
>
> **License:** MIT

This conversion reformats the original project into the Talent Market template structure (`profile.yaml`, `skills/`, `tools/`) following the [vibe-coding-guide](https://github.com/CarbonKite/talent-template/blob/main/vibe-coding-guide.md). All skill content and tool scripts are copied verbatim from the original repository.
